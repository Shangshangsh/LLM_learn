# 移动零

## 初步思路

题目让我们把全部的0移动到数组的末尾处，
这个就是很标准的两个指针啊，一头一尾来一个，然后开始，

过程是，如果l是0，r不是0，交换l和r，然后l++，r--，如果l不是0，r是0，那么r--，如果l和r都是0，那么只r--

```python 
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        l,r = 0,len(nums)-1
        while r >= l:
            if nums[l] == 0 and nums[r] !=0:
                nums[l],nums[r] = nums[r] ,nums[l]
                r -= 1
                l += 1
            elif nums[l] != 0 and  nums[r] != 0:
                l += 1
            elif nums[r] == 0:
                r -= 1
        return nums
```

我去，写完才发现题目要我们保持原来非0顺序不变

那应该用一个快慢指针，让快的一直指向非0元素，慢的指向0元素，

开始，各自找到自己的初始位置，然后快慢交换，慢的不要动，快的一直往后指

```python 
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        slow,fast = 0,1
        while fast < len(nums):
            #如果慢的为0快的不为0，交换，然后都++
            if nums[fast] != 0 and nums[slow] == 0:
                nums[fast],nums[slow] = nums[slow],nums[fast]
                fast +=1
                slow += 1
            #如果慢的不为0，快的为0，就继续
            elif nums[fast] == 0 and nums[slow] !=0:
                fast +=1
                slow += 1
            #如果都是0，只让快的+
            elif nums[fast] == 0 and nums[slow] == 0:
                fast += 1
            #如果都不是0，那么就都加
            elif nums[fast] != 0 and nums[slow] != 0:
                fast += 1
                slow +=1
        return nums
```
注意，一定要用elif，不要全用if！，否则会报错
