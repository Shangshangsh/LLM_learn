# 旋转图像

力扣48

# 初步思路

题目要求，把一个n*n的矩阵进行转置

emmm，是顺时针旋转90度

对于i j，

第一列去了第一行
第二列去了第二行

所以第i行去了第i列

竖着看：

第一列的元素去到第一行。
第二列的元素去到第二行。
……
第 j 列的元素去到第 j 行。
横着看：

第一行的元素去到最后一列。
第二行的元素去到倒数第二列。
……
第 i 行的元素去到第 n−1−i 列。（i 从 0 开始）
所以位于 i 行 j 列的元素，去到 j 行 n−1−i 列，即 (i,j)→(j,n−1−i)。

两次翻转等于一次旋转
(i,j)→(j,n−1−i) 可以通过两次翻转操作得到：

(i,j) 
转置
​
 (j,i) 
行翻转
​
 (j,n−1−i)


## 答案思路

最终我们需要拆解成两个部分

第一个先转置，转置的话直接弄成交换i j对应的元素就行

第二部反转，即矩阵左右反转过来就行



```python 
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        m,n = len(matrix),len(matrix[0])
        #先转置
        for i in range(m):
            for j in range(i+1,n):#只需要遍历到对角线的就行
                matrix[i][j],matrix[j][i] = matrix[j][i],matrix[i][j]
        
        #在反转，即每一行反转过来就行
        for row in matrix:
            row.reverse()
        


```
