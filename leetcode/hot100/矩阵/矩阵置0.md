## 初步思路

题目要求对于一个矩阵，如果一个元素是0，就把所在行和列都弄成0

emmm
这个感觉需要先把位置是0的给找出来，看看哪一行和那一列需要变成0就行了，然后统一操作


```python 
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        m,n = len(matrix),len(matrix[0])
        raw = set()#需要变成0的行
        col = set()#需要变成0的列
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:#如果遇到一个0，就把对应的行列加入
                    raw.add(i)
                    col.add(j)
        #简单粗暴，再遍历一遍
        for i in range(m):
            for j in range(n):
                if i in raw or j in col:#任何一个维度在斩杀内就要变成0
                    matrix[i][j] = 0
        return matrix
```

## 进阶

更好的做法是，直接使用matrix自己的行和列来存，不用一个set，即使用第一行和第一列来去设置

```python 
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        m,n = len(matrix),len(matrix[0])
        raw = set()#需要变成0的行
        col = set()#需要变成0的列
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:#如果遇到一个0，就把第一行和第一列对应的位置都弄成0
                    matrix[0][j] = 0
                    matrix[i][0] = 0
        #简单粗暴，再遍历一遍
        for i in range(m):
            for j in range(n):
                if matrix[0][j] == 0 or matrix[i][0] == 0:#去找对应的
                    matrix[i][j] = 0
        return matrix
```

这里有一个问题，如果说第一行本来就有一个0，我们记录，即记录了00，但是在遍历的时候直接被斩杀了，最后全部会变成0
