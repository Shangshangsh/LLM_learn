# 搜索二维矩阵

力扣240

## 初步思路

题目要求我们设计一个算法，在二维矩阵中搜索一个目标值

矩阵满足每一行都是升序，每一列都是升序


## 答案思路

将整个矩阵逆时针旋转45度去看，正好是一个二分搜索树！！

换句话说，我从第一行最后一列开始看，如果发现比target小，说明第一行全部都比target小，此时直接斩杀第一行，如果发现这个元素比target还要大，那么直接斩杀该元素所在的列，因为后面的列也是比他要大的

整理一下，用数模拟就是，如果比target大，行-1，如果小，就列-1，正好的话返回True


```python 
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        i,j = len(matrix)-1,0#从最右上角开始
        while i >=0 and j < len(matrix[0]):
            if matrix[i][j] > target:
                i -=1
            elif matrix[i][j] <target:
                j +=1
            else:
                return True
        return False
```
