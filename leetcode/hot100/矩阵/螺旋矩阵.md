# 螺旋矩阵

力扣54

## 初步思路

题目给定一个矩阵，要求返回顺时针的遍历顺序

1 2 3

8 9 4

7 6 5

## 思路

准备四个变量，分别是left，right，top，bottom

然后我们遍历，严格执行右下左上

比如右，就是从left到right，下就是从top到bottom，左就是从right到left，上就是从bottom到top

然后每一次遍历的时候，缩一下就行


```python 
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        m,n = len(matrix),len(matrix[0])
        left,right = 0,n-1
        top,bottom = 0,m-1
        ans = []

        while left<= right and bottom >= top:
            #先从右到左
            for i in range(left,right+1):
                ans.append(matrix[top][i])
            #弄完记得顶-1
            top+=1

            #然后是从上到下
            for i in range(top,bottom+1):#这里的top第一个在刚才的右到左的最后一个就加上去了
                ans.append(matrix[i][right])
            right-=1


            #从右到左,这里要判断一下，因为刚才已经改变了top和right，所以如果违背了的话，就不行
            if top <= bottom:
                for i in range(right,left-1,-1):
                    ans.append(matrix[bottom][i])
                bottom -=1

            #从下到上
            #这里也是
            if left <= right:
                for i in range(bottom,top-1,-1):
                    ans.append(matrix[i][left])
                left += 1
        return ans 
                

```
