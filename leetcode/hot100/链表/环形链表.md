# 环形链表

力扣141

## 初步思路

判断链表是不是有环的，思路是快慢指针

首先，由于有环，所以准备一个快慢指针开始冲，快指针一次跑两个，慢指针一次走一个

如果有环的话，那么这俩都不能为None，并且肯定是在环中

```python 
class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        if not head or not head.next:#为空或者只有一个节点
            return False
        slow,fast = head,head.next
        while slow != fast:#这里是不等于，所以上面的起跑线一定要错开
            if not fast or not fast.next:#fast跑的快，只判断fast就行
                return False
            slow = slow.next
            fast = fast.next.next
        return True

```
