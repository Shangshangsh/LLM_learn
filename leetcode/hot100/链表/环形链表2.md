## 初步思路

相交于1来说，这里需要返回出节点

那么流程是这样的，首先依旧是fast和slow，然后如果出现了None，直接return
false，如果说出来了，此时fast=slow，然后我们固定住fast不变，然后slow从头开始，两人都是一下一步，当它们再次相遇的时候，就是最最终的节点答案

```python 
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return None

        slow,fast = head,head
        while slow and fast:
            if not fast or not fast.next:
                return None
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                break
        #然后让slow回到head
        slow = head
        while slow != fast:
            slow = slow.next
            fast = fast.next
        return slow
```

写的不对，有两点需要注意，一个是开始的时候需要一起从同一个起点开始，否则会在数学上出问题，其次第二个阶段需要先判断，然后再走

```python 
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return None

        slow,fast = head,head
        while slow and fast:
            if not fast or not fast.next:
                return None
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                break
        #然后让slow回到head
        slow = head
        while slow and fast:
            if slow == fast:
                return slow
            slow = slow.next
            fast = fast.next
```
