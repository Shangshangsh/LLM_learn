## 初步思路

首先链表肯定是不能从后往前遍历

所以思路是，可以先把链表反转过来，然后从头节点和

哎题目要求我们用On的复杂度和O1的空间复杂度

既然链表无法反转，但还是需要判断，那么思路是，首先找到链表的一半，然后把后面一半的链表反转，这样就实现了从前往后，从后往前遍历了，

找到链表的一半，可以用快慢指针，即fast每次走两步，然后slow每次走一步就行

```python 
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        #首先用快慢指针找到mid
        slow,fast = head,head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        #然后这里的slow就是第二段的起点，然后开始反转

        #反转slow往后的
        pre,_next = None,None
        cur = slow
        while cur:
            _next = cur.next
            cur.next = pre
            pre = cur
            cur = _next
        #然后现在的链表是一个pre从后，一个head在前
        while head and pre:
            if head.val != pre.val:
                return False
            head = head.next
            pre = pre.next
        
        return True



```
