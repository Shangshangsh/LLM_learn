## 初步思路
反转链表一共要用到三个变量，分别是pre，next，cur

思路是，首先是先把next存下来，防止丢失

然后是把cur.next指向pre

然后把pre弄到cur上

然后cur跳到下一个（跳到next），

```python 
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        cur = head
        pre,_next = None, None
        while cur:
            _next = cur.next
            cur.next = pre
            pre = cur
            cur = _next
        return pre


```
