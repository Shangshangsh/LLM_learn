# 相交链表

## 初步思路

找到两个链表的相交的位置

这个简单，首先两个头开始遍历，统计一下长度，当一个遍历到结束的时候，另一个还没遍历完，此时算出差值，然后直接都从头开始，相差那个差值开始遍历，当两个相等的时候就是 return

如果一直没有return，说明就没有相交，直接return空



```python 
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        #首先先扫一遍，找到长的
        longer,short = headA,headB
        longer_length = 1
        short_length = 1

        while longer:
            longer = longer.next
            longer_length += 1
        while short:
            short = short.next
            short_length += 1
        longer = headA if longer_length >=short_length else headB
        short = headB if longer_length >=short_length else headA
        #然后需要去弄一个差值
        dif = abs(longer_length-short_length)
        while dif>0:#让长的走到对齐
            longer = longer.next
            dif -=1
        while longer and short:
            if longer == short:
                return longer#这里是要返回
            longer = longer.next
            short = short.next
        return None

```
