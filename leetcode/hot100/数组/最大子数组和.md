# 最大子数组和

力扣53

## 初步思路
题目要求，给了一个数组，问这个数组的子数组和最大是多少

所谓子数组，就是说，需要是连续的，求和最大就行了

这个依稀记得使用dp或者贪心来写，先用dp吧，定义dp[i]是以自己为下标结尾的子数组的长度

这里就是依赖dp[i-1]就行，换句话说，如果dp[i-1]+nums[i]还会变小的话，或者说比自己还小，那么干脆以自己作为起点就行了

所以递推就是 dp[i] = max(dp[i-1]+nums[i],nums[i])


```python 
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        if len(nums) ==1:#边界条件，如果只有一个，返回自己就行
            return nums[0]
        n = len(nums)
        dp = [0]*n
        dp[0] = nums[0]

        for i in range(1,n):
            dp[i] = max(dp[i-1]+nums[i],nums[i])
        return max(dp)
        
```
