# 合并区间

力扣56

## 初步思路

这题说，给了一个数组，数组里面是小数组，然后需要吧重复的合并

比如说intervals = [[1,3],[2,6],[8,10],[15,18]]

合并后就是

[[1,6],[8,10],[15,18]]

这个和那个打气球差不多，首先是要先排序

排序完后，直接开始遍历就行，从第二个开始遍历，如果第二个的0是小于第一个的1的时候，说明有重合，直接让第二个变成 【第一个的0，第二个的1】
然后继续遍历，直到不满足条件的话，让上一个加入到ans中就行

```python 
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        #先排序
        intervals.sort(key = lambda x: x[0])
        ans = []
        n = len(intervals)
        for i in range(1,n):
            if intervals[i-1][1]>= intervals[i][0]:#上一个的尾巴是大于当前的头，说明可以合并
                intervals[i] = [intervals[i-1][0],max(intervals[i][1],intervals[i-1][1])]#直接改当前的,注意右边界选择最大的
            else:#说明遇到不能合并的了，直接记录答案，记录i-1的就行，说不定后面自己的也会
                ans.append(intervals[i-1])
        #最后记录一下最后一个
        ans.append(intervals[-1])
        return ans
```

有个bug，就是如果i-1完全包住了i，那么右边界还是i-1的，所以是max一下啊就行
