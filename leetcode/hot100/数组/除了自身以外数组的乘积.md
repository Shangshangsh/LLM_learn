# 除了自身以外数组的乘积

力扣238

## 初步思路

给定一个数组，题目要求我们返回这个数组ans

对于ans[i]的含义是，除了i本身以外，其他所有元素的乘积

说白了，直接先把数组遍历一遍都乘，然后再除以nums i就行了，但是题目要求不能使用除法，并且需要On的复杂度



emmmm

[1,2,3,4]

既然不让用除法，那就只能，看看能不能仿照前缀和的形式，搞出前缀积

前缀积就是
1 2 6 24

后缀积就是
24  24 12 4

这样好像不行，因为前缀和后缀都带着自己，最好是弄成i的前缀积只含有0到i-1位置上的，但是这样的话0位置要弄成谁，算了先弄成-1试试

-1 1 2 6

24 12 4 -1

这样的话，直接拿前缀乘后缀就行哈哈哈，如果前后缀有一个是-1的话，直接取另一个就行

或者说，直接用自身就行，因为只有首和尾是特殊的

```python 
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        pre = [0]*n#前缀积
        _next = [0]*n#后缀积

        #最初始化
        pre[0] = nums[0]
        _next[-1] = nums[-1]

        #先构建前缀积
        for i in range(1,n):
            if i == 1:#第二个是直接继承第一个
                pre[i] = pre[0]
            else:
                pre[i] = pre[i-1]*nums[i-1]

        #构建后缀积
        for i in range(n-2,-1,-1):
            if i == n-2:#倒数第二个也是继承最后一个
                _next[i] = _next[-1]
            else:
                _next[i] = _next[i+1]*nums[i+1]
        #0位置的直接就是后缀和的第一个，n-1位置的是前缀的第一个，其他都是前缀乘后缀
        ans = [0]*n
        ans[0] = _next[0]
        ans[-1] = pre[-1]
        for i in range(1,n-1):
            ans[i] = pre[i]*_next[i]
        return ans 
```
