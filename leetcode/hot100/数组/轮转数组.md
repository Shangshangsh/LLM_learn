## 初步思路

题目说，给我们一个数组，要求将数组中的元素向右轮转k个位置

什么叫做轮转k个位置，例子是

1 2 3 4 5 6 7

k = 3

就是说，0位置移动到1位置，2位置移动到3位置，以此类推，最后变成

7 1 2 3 4 5 6
这就算轮转一次，那么轮转3次就是

5 6 7 1 2 3 4


emmm，想了一下，在轮转k次后，对于i位置，那么他应该出现在

比如0位置，轮转三次就是往后移动三次，那么他就应该出现在3位置，即i+3

那如果不够怎么办，比如6位置的7，轮转3次后变成了下标9，

如果我用9%7，余2，那么应该出现在2位置，所以就是（i+k）%n的位置

知道了各个位置在轮换后的具体位置了，那么下一步就是怎么转移过去

一个比较简单的做法是，直接新开一个长度也是n的数组，直接copy过去就行

```python 
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        ans = [0]*n
        for i in range(n):
            if i+k<=n-1:#正常的没过界
                ans[i+k] = nums[i]
            else:#过界了
                ans[(i+k)%n] = nums[i]
        return ans 
```

## 全部反转

注意到，我把数组全部反转后是

7 6 5 4 3 2 1

注意到，在这里，我可以划分的反转
即前面的7 6 5反转为 5 6 7

然后后面的4 3 2 1 反转为 1 2 3 4

换句话说，我先按照k = 3劈开，变成了

1 2 3 4和 5 6 7其实就把5 6 7移动到前面去了，内部并没有改变，


再比如k = 4，那么就是1 2 3 4 5和 6 7

然后变成 6 7 1 2 3 4 5，然后反转变成 7 6 5 4 3 2 1

哎不对，不是，

其实k的意思是，将后面的k个元素反转到前面去


所以最后的思路是，首先全部反转一遍

这个后面k个就变成前面k个了，

然后前k个和后n-k个在各自内部反转就行


```python 
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        #先整体反转一下
        nums = self.reverse(nums,0,n-1)

        #然后反转前k个和后n-k个

        nums = self.reverse(nums,0,k-1)

        nums = self.reverse(nums,k,n-1)
    

    def reverse(self,nums,i,j):
        while i<j:
            nums[i],nums[j] = nums[j],nums[i]
            i+=1
            j-=1
        return nums
```

注意，如果k大于n的话是有问题的

比如n是2，k是3

但其实，如果反转了n次，就回到了原点其实，所以加一个k = k%n就行
