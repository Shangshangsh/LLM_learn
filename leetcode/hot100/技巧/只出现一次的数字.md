# 只出现一次的数字

## 初步思路

这道题说，我有一个数组，只有一个数字是只出现一次的，其他的数字都是出现了好多次
问我们只出现一次的那个是谁

只能用O1的空间复杂度以及On的时间复杂度

这题的做法是用异或运算，即相同为 0，不同为 1

就是任何数，与0 进行 异或运算，得到的还是本身

与本身异或运算，得到的结果是0

并且满足交换律

所以说，我们轮流来进行异或，这样相同的就会被斩杀，不相同的那个最后运算完就会被留下来

在python中，异或运算就是^就可以了

```python 
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        x = 0
        for num in nums:
            x = num^x
        return x
```
