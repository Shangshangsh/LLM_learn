# 多数元素

## 初步思路

题目要我们去找到出现次数多于n/2的元素（下取整），最简单的方法就是遍历一遍统计，然后再遍历一下，这样时间复杂度是ON，空间复杂度也是ON

## 摩尔投票
非常的骚猪

就是打擂台，准备俩变量，一个是count，一个是candidate

然后数字一个一个上去，如果count = 0，说明擂台还没有人，直接candidate变身

如果上来一个和candidate不一样的，直接让count-1

如果上来的是和cndidate一样的，那么擂台加人，

此外不用打担心减到0，因为count = 0有兜底

```python 
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        count = 0
        candidate = 0
        for num in nums:
            if count == 0:
                candidate  = num
                count += 1
            else:
                if num == candidate:
                    count += 1
                else:
                    count -=1
        return candidate
        
```
