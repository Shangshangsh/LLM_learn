# 寻找重复数

## 初步思路

这题说，给了一个数组，数组长度是n，然后数组有n-1个自然数，那么肯定是有一个数重复了，我们就是要找到那个重复的数

要求是 On O1


这题very very的巧妙

把数组想象成一个链表！！把下标看成是一个节点，把下标里面的值看成是next

比如[1, 3, 4, 2, 2]

在下标0出发，指向1，到3，然后指向3，指向2 ，然后到2，然后指向2到4 然后指向4 到2，然后指向2到4，然后指向4到2

nums[0] = 1，下一步走到下标 1。

nums[1] = 3，下一步走到下标 3。

nums[3] = 2，下一步走到下标 2。

nums[2] = 4，下一步走到下标 4。

nums[4] = 2，下一步走到下标 2。

有没有发现，这样就形成一个环了，为什么会出现这样，因为有重复的下标（3和4）的值是一样的！

所以这就是环形链表2一样的了，首先一个快慢指针，然后他们早晚会相遇，等相遇后，再让一个从起点走就行

```python 
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        fast,slow = 0,0
        #第一步，先找入口，最后如果有环的话，就是fast==slow就是
        while True:
            slow = nums[slow]
            fast = nums[nums[fast]]
            if slow == fast:
                break
        slow = 0
        while True:
            slow = nums[slow]
            fast = nums[fast]
            if slow == fast:
                return slow


```
