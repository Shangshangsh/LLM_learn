# 颜色分类

## 初步思路

这题说是，我们的数组有三种颜色 0 1 2，然后需要吧颜色相同的都放在一起

这题就是荷兰国旗本尊

但现在要求我们依旧是用On的时间复杂度，O1的空间复杂度

## 思路
我们维护三个指针，分别是p0，代表着下一个0应该存放的地方 （初始在0

p2 代表下一个2应该存放的地方（初始值在n-1

i当前遍历的指针，从0开始

我们开始遍历

首先如果i遇到了0，那么我们就交换i和p0，这样p0位置就变成了0，自然p0就要++，此外，由于i是走在p0前面的，所以说从后面换到前面的其实就是1（因为i和p0同时开始，只有遇到0才会换）

如果遇到了1，就应该呆在中间不动，然后i++

如果遇到了2，那么直接把2换到p2，然后p2--，注意此时i不能动了，因为不知道从尾巴换回来的是谁，所以i不能动

```python 

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        p0 = 0
        p2 = n-1
        i = 0
        while i <= p2:#注意这里，要写成p2，即换到p2就好了，不然写成n的话，就会把p2后面的也打乱
            if nums[i] == 0:
                nums[p0],nums[i] = nums[i],nums[p0]
                p0 += 1
                i +=1
            elif nums[i] == 1:
                i+=1
            elif nums[i] == 2:
                nums[p2],nums[i] = nums[i],nums[p2]       
                p2 -= 1
```
